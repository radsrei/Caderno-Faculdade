Informações sobre o Comandos de Arquivos


Comando Descrição Imagem de exemplo
MKDIR Cria pastas Comando Mkdir
RMDIR Apaga pasta quando vazia Comando Rmdir
CP Saida do Terminal do Linux Comando Cp
TOUCH Mostra o caminho que se encontra para chegar a posição atual Comando Touch
CAT Limpa o terminal deixando na primeira linha Comando Cat
MV Move o arquivo entre pastas
